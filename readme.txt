WooCommerce extension to hide product price and add to cart button for guest customers, forcing them to get registered with you.

== Description ==

**Woo Login To See Price And Buy** an WooCommerce extension to hide product price and add to cart button for guest customers, forcing them to get registered with you.

= Features =

* **Hide Product Price**
Hides product price for guest customers. Setting to enable/disable this feature.
* **Hide Product Add To Cart Button**
Hides product add to cart button for guest customers. Setting to enable/disable this feature.
* **Forces Guest Customers To Get Registered With You**
By hiding product price and add to cart button for guest customers, you can force them to get registered with you.
* **Custom Message In Place Of Product Price For Guest Customers**
In addition to hide price for guest customers you can also add an alternative text to them, giving them message to get registered with you.
* **Custom Message In Place Of Product Add To Cart Button For Guest Customers**
In addition to hide add to cart button for guest customers you can also add an alternative text to them, giving them message to get registered with you.
* **Customize Login Link**
You can make a customized login link with custom url and text to be used when you hide price and add to cart button for guest customers.
* **Highly Customizable Settings**
So much settings are available to customize the usage of plugin, like custom text to show to user, render your own html, set your own color for text, etc.
* **Highly Intuitive Plugin**
Plugin is so much user-friendly that you must fall in love with it.
 
== Installation ==

= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of "Woo Login To See Price And Buy", log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Woo Login To See Price And Buy" and click Search Plugins. Once you've found our "Woo Login To See Price And Buy plugin" you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =
The manual installation method involves downloading our "Woo Login To See Price And Buy" plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
